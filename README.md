# La Vie en Rose

A responsive Angular 21 LTS single-page application for a modern nail salon.

## Development

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Production build

```bash
npm run build
```

Place final salon photography in `public/assets/images` and update the image
references in the page templates. The current visual placeholders are made with
CSS so the site remains self-contained while assets are being prepared.
